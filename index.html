AI Document Processing & Filing System

Client automation · 2 zaps, 39–40 steps each

A dual-zap document intelligence pipeline that watches Google Drive upload folders for new invoices and bank statements. Each file is converted to text, read by Gemini AI to extract structured data (client names, vendors, bank names, dates), then automatically renamed and filed into the correct nested folder. A second AI call matches extracted metadata to existing folder names, with branching logic to handle missing folders gracefully.

  • Gemini 2.5 Flash Lite reads raw document text and extracts structured fields — no templates or OCR rules needed
  • Second AI call (OpenAI) matches extracted data to the correct destination folder from a dynamic folder list
  • Auto-renames files using AI-extracted metadata (e.g., ClientName.Vendor.Date.pdf)
  • Every processed document is logged to Google Sheets for a complete audit trail
